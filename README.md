# AI Recommendation System

## Description
This is a simple AI Recommendation System developed using Python. It recommends Movies, Books, Music, or Games based on the user's choice.

## Features
- Recommends Movies
- Recommends Books
- Recommends Music
- Recommends Games
- User-friendly menu system

## Technologies Used
- Python 3

## How to Run
1. Open the Python file in VS Code.
2. Run the program.
3. Choose an option from 1 to 4.
4. Get recommendations based on your interest.

## Sample Output

=== AI Recommendation System ===

Choose your interest:
1. Movies
2. Books
3. Music
4. Games

Enter your choice (1-4): 1
